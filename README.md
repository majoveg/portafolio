## 👋 Sobre este proyecto

Este portafolio es más que una colección de proyectos: es una forma de pensar el diseño.  
Fue construido para mostrar cómo conecto la investigación con el diseño, el diseño con 
la estrategia, y la estrategia con resultados medibles.

Aquí encontrarás casos de estudio en:
- 📱 **Healthtech** — Aplicación móvil para mejorar la experiencia del paciente en urgencias
- 🎓 **E-learning** — Metodología de producción de cursos y plataformas educativas
- 🛍️ **SaaS B2B/B2C** — Diseño de producto y gestión en plataformas como YoProfe y MentorIA
